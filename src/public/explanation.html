ফায়ারবেস ফায়ারস্টোর সেটআপ গাইড

কিভাবে আপনার ফায়ারবেস প্রজেক্টে Cloud Firestore চালু এবং কনফিগার করবেন।

১

ফায়ারবেস কনসোল খুলুন

আপনার ওয়েব ব্রাউজারে Firebase Console খুলুন এবং আপনার প্রোজেক্টটি সিলেক্ট করুন।

২

Firestore Database এ যান

বাম দিকের মেনু থেকে 'Build' সেকশনের নিচে Firestore Database এ ক্লিক করুন।

বাম দিকের মেনু থেকে Firestore Database সিলেক্ট করুন।

৩

Create Database বাটনে ক্লিক করুন

যদি আপনার কোনো ডাটাবেস তৈরি করা না থাকে, তাহলে আপনি একটি 'Create database' বাটন দেখতে পাবেন। এটিতে ক্লিক করুন।

'Create database' বাটনে ক্লিক করে শুরু করুন।

৪

Test Mode নির্বাচন করুন

আপনাকে Production mode অথবা Test mode এর মধ্যে একটি বেছে নিতে বলা হবে। অ্যাপটি ডেভেলপ করার সময় এবং সহজে ব্যবহারের জন্য Start in test mode নির্বাচন করুন।

গুরুত্বপূর্ণ: Test mode আপনার ডেটাবেসকে ৩০ দিনের জন্য খোলা রাখবে। প্রোডাকশন অ্যাপের জন্য আপনাকে পরে নিরাপত্তা বিধি (Security Rules) আপডেট করতে হবে।

ডেভেলপমেন্টের সুবিধার জন্য Test mode নির্বাচন করুন।

৫

লোকেশন নির্বাচন করুন

আপনার ব্যবহারকারীদের কাছাকাছি একটি লোকেশন নির্বাচন করুন। সাধারণত, ডিফল্ট লোকেশনটিই যথেষ্ট। এরপর 'Enable' বাটনে ক্লিক করুন।

আপনার পছন্দমতো একটি লোকেশন নির্বাচন করুন।

৬

নিরাপত্তা বিধি (Security Rules) আপডেট করুন

ডাটাবেস তৈরি হয়ে গেলে, উপরের দিকে Rules ট্যাবে ক্লিক করুন।

সেখানে থাকা লেখাগুলো মুছে দিয়ে নিচের কোডটি পেস্ট করুন:

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

এরপর Publish বাটনে ক্লিক করে পরিবর্তনটি সেভ করুন।

Rules ট্যাবে গিয়ে কোডটি পেস্ট করে Publish করুন।
