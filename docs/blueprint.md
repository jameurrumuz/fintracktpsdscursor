# **App Name**: FinTrack

## Core Features:

- Transaction History Display: Displays the transaction history in a table format, including date, type, method, amount, and description.
- New Transaction Input: Allows users to add new transactions with details such as date, description, amount, payment method, and type.
- Balance Summary: Provides a summary of balances, including cash, bank, and total balances, updated in real-time based on transactions.
- Data Backup and Restore: Enables users to backup and restore transaction data to prevent data loss.
- Transaction Filtering: Allows users to filter transactions by type, payment method and date.
- Smart Categorization: Suggest categories, using a tool, to classify a transaction.

## Style Guidelines:

- Primary color: Deep sky blue (#3A86FF) for a clean, trustworthy feel.
- Background color: Light gray (#F5F5F5) to ensure readability and reduce eye strain.
- Accent color: Soft violet (#8B80F9) as a highlight for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, known for its modern and neutral design that ensures excellent readability across various screen sizes.
- Simple, outline-style icons to represent transaction types and actions.
- Clean, card-based layout for balance summaries and transaction inputs to provide a structured user experience.
